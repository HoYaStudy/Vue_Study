# Vue_Study
This repository is for HoYa's Vue study.

## Install Vue-CLI
> $ yarn global add @vue/cli

## Create Project with Vue-CLI
> $ vue create <project_name>

## Use TailwindCSS
> $ vue add tailwind

## Run
> $ yarn serve

> $ yarn serve -- --port <port_number>

## Build
> $ yarn build
